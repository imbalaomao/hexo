【科学上网】跨越长城，走向世界

发表于2023-08-03|更新于2023-11-01|科学上网
|阅读量:

前言

自2012年接触翻墙以来，我大约用过以下各类软件：

  • VPN系列：PPTP、L2TP类软件（类似GreenVPN等）
  • 法轮功成员开发软件：Lantern（蓝灯）、自由门、无界……（可能还有，但我就用过这几种）
  • 初代代理软件：Psiphon（赛风）、XX-NET（基于GAEAgent且自带一定混淆的软件，直至今天仍在更新）
  • 传奇代理软件：Shadowsocks(R)（可以说没有这款软件就没有今天的代理软件百家齐放的局面。R.I.P. Clowwindy&Breakwa11）
  • 现代代理软件：v2ray、xray、Clash、QuantumultX、Shadowrocket、Surge…
  • 小众代理软件：Tuic、Hysteria、Naiveproxy、mieru…

在这分享（记录）一下，方便后续查阅。

软件归类

各类核心

请注意：本处仅存放平时本人可能会用到的核心，以外的完全不涉及

  • v2ray/v2fly
  • xray
  • Clash Foss
  • Clash Premium
  • Clash Meta
  • Sing-box
  • NaiveProxy
  • Hysteria
  • Tuic
  • Juicity(Tuic魔改)
  • Mieru

Windows GUI

Clash系

  • Clash for Windows（软件较大，功能齐全，方便调试）
  • Clash for Windows汉化版
  • Clash Verge（需要EdgeRuntime2，轻量好用）
  • Fclash（使用Flutter编写的跨平台GUI）
  • ClashN（V2rayN作者做的同界面客户端）
  • Clash.Mini（使用Go语言编写的简洁版UI，集成了Meta内核）
  • EasyClash（不良林编写，仅包含基本设置及核心，适合极简强迫症使用）
  • ClashWarden（使用C++编写的小巧客户端）
  • ClashR Auto（基于CFW魔改，更适合小白使用）
  • clash-for-flutter（使用谷歌Flutter框架编写）
  • Clashy（桌面全平台）

V2ray系

  • Nekoray（魔改Qv2Ray，增加hysteria等新协议支持）
  • V2rayN（已使用.NET 6.0更新UI）
  • Qv2ray（原维护者弃坑，备份在此）

Sing-Box

  • Sing-box 托盘GUI

iOS/MacOS GUI

  • Shadowrocket(仅限iOS系统，不支持intel Mac，一般定价：2.99USD)
  • QuantumultX(iOS/MacOS授权通用，支持Intel-Mac/M1-Mac，一般定价：4.99USD)
  • Surge(iOS/MacOS授权分离，定价超高，一般不推荐)
  • Loon(iOS/MacOS授权通用)
  • Stash(iOS可用，2.99USD)
  • ClashX（仅限Mac使用）
  • ClashX Meta（仅限Mac使用）
  • V2rayX（仅限Mac使用）
  • Sing-box 托盘GUI（仅限Mac使用）

Android GUI

V2ray系

  • SagerNet（兼容SSR、v2ray、clash配置，带很多插件）
  • Matsuri（魔改版Sagernet，与原版差距不大，兼容原版插件）
  • V2rayNG（Xray社区钦定）

Clash系

  • Clash for Android
  • Clash Meta for Android
  • Clash for Magisk（需ROOT）

Surfboard（兼容Surge4配置文件）

搭建脚本归类

所有脚本我本人都使用过，无好坏之分，只有适合你自己的才是最好的

v2ray/xray脚本

  • 233boy经典脚本
  • 233boy的xray一键脚本（推荐）
  • mack-a 编写的八合一脚本，简单易用（目前本人正在使用）
  • 1024.day站长自编简易wss+ss安装脚本，脚本足够安全
  • 基于Sing-box的Vless+Reality节点搭建脚本
  • vless+XTLS_VISION/gRPC+REALITY 手动搭建方法

Trojan脚本

Trojan存在明显且可探测的 TLS in TLS 特征，因此不推荐自建使用

小众协议脚本/教程

本Part中很少会出现一键脚本，大多为分步骤教学，需要对Linux有一定基础知识。同时大多数小众协议并无图形化UI，因此请使用v2rayN或Nekoray添加自定义核心后使用，具体使用方法请参考此处

  • Hi-Hysteria一键安装脚本
  • Tuic安装指南（目前本人主要使用）
  • Naiveproxy一键安装脚本
  • mieru官方教程（分步骤且为英文，非脚本）
  • socks5代理搭建

小众协议补充：

  1. 目前对于如Racknerd、virmach这类无药可救的机器，可以使用Hysteria协议，如有CN2GIA、CU4837、CU9929等优秀线路的机器推荐优先考虑Tuic、Naiveproxy等有流控的协议（Hysteria搭建在Racknerd机器上易断流）。
  2. Clash.meta目前已对Tuic、Hysteria做了较为完善的支持，不想添加自定义配置和核心的可以使用此软件（关于替换核心可以看本人教程）

面板搭建

  • 原版X-UI（很久不更新了）
  • 魔改X-UI

VPS测试脚本

  • bin大佬的DD重装脚本（使用官方重装，非第三方DD镜像）
  • 萌咖大佬的DD脚本（如VPS为甲骨文ARM，则需要加 -firmware）
  • 宝塔面板开心版7.7.0（无后台监控）
  • 集大成一键测试脚本
  • 常规解锁测试
  • 开启 BBR + fq
  • BBR 脚本（KVM架构）
  • BBR 脚本（OpenVZ架构），需要开启TUN/TAP
  • 测速脚本
  • 回程测试
  • Warp脚本

机场

对于小白且对于翻墙IP要求低的用户来说，机场是最好的选择，以下推荐的都是本人现在在用的机场，可以放心购买：

  • YToo（歪兔：极稳定、贵、速度很快）
  • 大航海（自带超高质量Emby服务器）
  • X-air（稳定、快速、便宜）
